### 344.[反转字符串](https://leetcode.cn/problems/reverse-string/description/)

这题挺简单的直接写代码

```java
class Solution{
	public void reserveString (char []s){
		int l =0;
    int r =s.length -1;
    while(l<r){
      char tmp =s[l];
      s[l]=s[r];
      s[l]=tmp;
      l++;
      r--;
    }
  }
}
```
